There are many ways to use svg in React. We can directly use the svg as src in the img tag or we can create custom component to render the svg as paths in html. In this article we will create a simple React app using svg image.

Whats special about SVG with React

At first it might feel a trivial task. But if you want interaction with svg elements, you need to create custom component so that the svgs are rendered as path and not just an image.

There are multiple ways to add svg to react components.

  • Using <img> tag Just import the svg file and set the src of the img to the imported file. With this approach you will get the file rendered but it will render as plain img. You would not be able to utilize the svg elements.
import Map from './india.svg';

function App() {
  return (
    <div className="App">
      <img src={Map} alt="indiamap"/>
    </div>
  );
}

export default App;

  • You can directly use svg images out of box with Create React APP 2.0. Just import the svg image with this predefined syntax. The webpack config for the CRA does all the work for you behind the scene. As a result, you don’t need to run around for plugins for simple use cases.

  • Using svg custom component. For this to work you need to convert your svg file into a React component. If its a small file, you can convert the file manually otherwise you can use svgr utility.
